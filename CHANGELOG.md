# Changelog

All notable changes to the Eheca Framework will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]
### Added
- Support for PHP 8.4
- GitHub Actions workflow for CI/CD
- Code coverage reporting with Codecov
- Enhanced security features
- Performance optimizations for PHP 8.4

### Changed
- Updated minimum PHP version requirement to 8.4
- Upgraded Symfony components to latest stable versions
- Improved error handling and logging
- Updated development dependencies

### Fixed
- Resolved compatibility issues with PHP 8.4
- Fixed deprecation warnings
- Security vulnerabilities in dependencies

## [1.0.0] - 2025-05-18
### Added
- Initial release of Eheca Framework
- Core framework components
- Basic routing and controllers
- Twig template engine integration
- Database abstraction with Doctrine DBAL
- Authentication system with JWT
- Console commands
- Testing infrastructure
